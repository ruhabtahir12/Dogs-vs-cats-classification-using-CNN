# Dogs-vs-cats-classification-using-CNN
CNN-based Dogs vs Cats image classifier built with TensorFlow/Keras, achieving ~81% validation accuracy with good generalization.
This project is a Deep Learning-based Image Classification system that distinguishes between images of dogs and cats using a Convolutional Neural Network (CNN). The model learns visual patterns such as shapes, textures, and features from images to make accurate predictions.
Model Architecture

The CNN model includes:

Convolutional Layers (feature extraction)
Activation Function: ReLU
Max Pooling Layers (downsampling)
Fully Connected (Dense) Layers
Output Layer with Sigmoid Activation (Binary Classification)
